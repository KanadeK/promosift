# PromoSift

> 将杂乱的游戏截图文件夹整理成干净、多样、适合商店展示的截图集——全部在浏览器本地完成。

[English](README.md)

PromoSift 提供分辨率、比例、曝光、对比度、近空白、重复图和视觉多样性等技术性提示；选择始终由你手工完成。

**PromoSift 不会上传、修改或删除你的截图。**

**质量分数属于技术性启发式提示，不代表审美评价。**

深色、低对比度或视觉相近的截图仍可能是正确的创意选择。发布前请始终人工复核。

支持 PNG、JPEG、WebP；支持选择文件、选择文件夹、拖放和从剪贴板粘贴。导出 ZIP 保留选中原图的原始字节，并附带 Contact Sheet、CSV 和不含图片数据的项目 JSON。

本地开发：`npm ci && npm run fixtures && npm run dev`。完整检查：`npm run check && npm run test:e2e`。

# Privacy

PromoSift is a static site. Import, decoding, analysis, duplicate grouping, contact-sheet generation, and ZIP creation run in the browser. It has no server, account, telemetry, database, cloud sync, remote font, or advertising dependency.

The exported project JSON does not include image data or absolute device paths. Images are accessed through browser-provided File objects and temporary object URLs, which are revoked on reset. An export includes only the images you mark Keep, plus local report metadata.

Use the browser’s normal download prompt to choose where exports are stored. You remain responsible for reviewing filenames and material before sharing.
